DESCRIPTION:

PMFormat is a diskette formatter with the following features:

    It has a graphical user interface
    It has a graphical presentation of defect sectors
    It has a quick format facility, that formats a diskette in only a few seconds
    It writes a special boot sector, that boots from hard disk, even if the diskette is in the drive and there is no kernel on it
